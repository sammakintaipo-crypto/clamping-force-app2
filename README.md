# Clamping Force Calculator — v4.2 Light（Kp vs 壓力法比較）

本版提供 **較淡色的淺色主題**，提升可讀性，同時保留：
- **Kp vs 壓力法比較**（差異%、一鍵同步 A/N/補償、依 Kp 反推 P）
- 材料搜尋、30 材料（Kp + 建議P）、單位換算、多模穴、薄壁/深度補償、噸位清單自訂、PWA
- Hard Align：兩句開關與子欄位始終靠左；RWD：桌面 32px、手機 24px

## 部署（GitHub Pages）
1. 建立公開 Repo：`clamping-force-app`
2. 上傳檔案至 Repo 根目錄
3. Settings → Pages：Deploy from a branch → main → /(root) → Save
4. 約 30–120 秒後上線
